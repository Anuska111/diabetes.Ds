# diabetes.Ds
🩺 Diabetes Prediction using Machine Learning

📌 Overview
This project predicts whether a patient is diabetic or non-diabetic using machine learning techniques. The model analyzes patient health parameters such as age, BMI, blood glucose level, HbA1c level, hypertension, heart disease, gender, and smoking history to make predictions.

##🎯 Objective
The objective of this project is to build an accurate machine learning model that helps in the early prediction of diabetes using patient medical records.

## 📂 Dataset
- "Source:" Kaggle Diabetes Prediction Dataset
- "Target Variable:"`diabetes`
  - 0 → Non-Diabetic
  - 1 → Diabetic
